# Sorteador de Números Aleatórios

Precisa sortear números aleatórios? Faça em segundos! 🔢✨

## 📝 Sobre o Projeto

Este projeto foi desenvolvido durante o curso de **Lógica de Programação** da Alura 🧠. Utiliza **JavaScript, HTML e CSS** para criar um sorteador de números aleatórios, onde você pode:

- Definir a **quantidade** de números a serem sorteados.
- Escolher o **intervalo** dos números.

## ✅ Status do Projeto

✔️ **Concluído!**

O sorteador está pronto para uso. Experimente e divirta-se!

## 💻 Tecnologias Utilizadas

- **HTML** – Estrutura da página 🖋️
- **CSS** – Estilização e layout 🎨
- **JavaScript** – Cérebro do projeto ⚙️

## 🚀 Como Usar

1. Abra o sorteador no navegador. 🌐
2. Escolha a **quantidade** de números a sortear. 🧮
3. Defina o **intervalo** (Ex.: 1 a 100). 📏
4. Clique em **"Sortear"** e veja o resultado! ✨
